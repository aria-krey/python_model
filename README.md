«Из-за сильного дисбаланса классов метрика Accuracy не отражает реальное качество моделей. Поэтому основное внимание уделяется F1-score (macro), который позволяет корректно оценивать качество классификации редких атак.»

«Для устранения сильного дисбаланса классов применялся алгоритм SMOTE, генерирующий синтетические примеры миноритарных классов только на обучающей выборке. Это позволило существенно улучшить macro-F1 и качество классификации редких атак.»

«После устранения дисбаланса классов с помощью SMOTE оценка моделей проводилась по метрике F1 macro, более корректно отражающей качество классификации при неравномерном распределении классов. Это позволило объективно сравнить модели и снизить влияние доминирующего класса.»
